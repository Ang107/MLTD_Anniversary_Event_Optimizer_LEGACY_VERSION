プライバシーポリシー

本サイト（以下「当サイト」）は、利用者のプライバシーを尊重し、取得する情報を適切に取り扱います。

運営者

当サイトは個人が運営しています。

取得する情報と利用目的

当サイトでは、利用状況の把握およびサービス改善のため、閲覧したページ、利用環境、概略の地域、アクセス元などのアクセス情報を取得します。これらの情報のみから、当サイト運営者が利用者個人を直接特定することはありません。

アクセス解析ツールについて

当サイトでは、利用状況の把握とサービス改善のため、以下の2つのアクセス解析ツールを使用しています。

Cloudflare Web Analytics

当サイトでは、Cloudflare, Inc.が提供する「Cloudflare Web Analytics」を使用しています。これはCookieを使用せず、利用者個人を特定したり、複数のサイトをまたいで追跡したりすることなく、ページの閲覧状況などを集計するアクセス解析ツールです。Cookieを使用しないため、この計測はすべての利用者を対象に行われます。

詳しくは、Cloudflareのプライバシーポリシーをご確認ください。

Google Analytics

当サイトでは、Googleが提供するアクセス解析ツール「Google Analytics」を使用しています。Google AnalyticsはCookieを使用してアクセス情報を収集するため、利用者が許可した場合にのみ計測を行います。収集される情報はGoogleのプライバシーポリシーに基づいて管理されます。

Googleによる情報の取扱いについては、Googleのサービスを使用するサイトやアプリから収集した情報のGoogleによる使用をご確認ください。

Cookieの無効化

ブラウザの設定でCookieを無効にするか、Google Analytics オプトアウト アドオンを利用することで、Google Analyticsによる計測を無効にできます。

すると、Google AnalyticsによるCookieを用いたアクセス解析の許可・拒否を選び直せます。なお、Cloudflare Web AnalyticsはCookieを使用しないため、この設定の対象外です。

第三者への提供

当サイト運営者は、法令に基づく場合を除き、利用者を直接特定できる情報を第三者へ提供しません。ただし、Google Analyticsによる情報の収集・取扱いについては、前項をご確認ください。
